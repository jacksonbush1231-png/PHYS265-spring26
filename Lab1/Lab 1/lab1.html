def phi (M,x,y,R):
    """
    This function returns the gravitational potential at a position OUTSIDE the radius of a body with mass
    
    arguments
    x and y (x and y coordinates of the evaluation point), float or np array, meters
    M (mass of the body), float, kg
    R (Radius of the body), float, meters

    Returns potential as a float or array
    """

    if type(M) is not float:
        raise (TypeError(f"Mass input must be float"))
    if type(R) is not float:
        raise (TypeError(f"R inputs must be floats"))
    if M<0 or R<0:
        raise (ValueError(f"Both radius and mass values must be positive"))
        
    if not isinstance(x,(float,np.ndarray)):
        raise (TypeError(f"x inputs must be arrays or float"))
    if not isinstance(y,(float,np.ndarray)):
        raise (TypeError(f"y inputs must be arrays or float"))
    
    
    r = np.where((np.sqrt(x**2+y**2))>R, np.sqrt(x**2+y**2), np.nan)
    G=6.67E-11
    V=-(G*M)/r
    assert isinstance (V,(float,np.ndarray)), "Potential is not a float or array"
    return V

Part 3: The Gravitational Force Field of the Earth-Moon System¶

The gravitational force $\vec{F}$ that a mass $M_1$ exerts on a mass $m_2$ is:

$$\vec{F}_{21} = -G \frac{M_1 m_2}{|\vec{r}_{21}|^2} \hat{r}_{21}$$

where $\vec{r}_{21}$ is the displacement vector from $M_1$ to $m_2$.

def F_21 (M1,m2,x1,y1,x2,y2,R1):

    """
    This function returns the gravitational force at a position OUTSIDE the radii of two bodies, with Body 1 being larger
    
    arguments
    M1 (Mass of Body 1), float, kg
    m2 (Mass of Body 2), float, kg
    x1 and y1 (x and y coordinates of Body 1), float or np array, meters
    x2 and y2 (x and y coordinates of Body 1), float or np array, meters
    R1 (Radius of Body 1), float, meters

    Returns Force as a float or array
    """
    if not isinstance(M1,(float,int)):
        raise (TypeError(f"Mass inputs must be floats or ints"))
    if not isinstance(m2,(float,int)):
        raise (TypeError(f"Mass inputs must be floats or ints"))
        
    if M1<0 or m2<0 or R1<0:
        raise (ValueError(f"Both radius and mass values must be positive"))
        
    if not isinstance(x,(float,np.ndarray)):
        raise (TypeError(f"x inputs must be arrays or float"))
    if not isinstance(y,(float,np.ndarray)):
        raise (TypeError(f"y inputs must be arrays or float"))

    G=6.67E-11

    rx=x2-x1
    ry=y2-y1

    r=np.where(np.sqrt(rx**2+ry**2)>R1,np.sqrt(rx**2+ry**2),np.nan)
    
    Fx_mag=-G*M1*m2/(r)**2
    Fx_dir=(rx)/np.abs(r) #definition of r hat x comp
    Fx=Fx_mag*Fx_dir
    assert isinstance(Fx,(float,np.ndarray)), "One F value is not a float or array"
    
    
    Fy_mag=-G*M1*m2/(r)**2
    Fy_dir=(ry)/np.abs(r)
    Fy=Fy_mag*Fy_dir
    assert isinstance(Fy,(float,np.ndarray)), "One F value is not a float or array"

    return Fx, Fy

Part 4: Altitude of the Saturn V Rocket¶

Rockets function by conservation of momentum. The ejection of fuel backwards propels the rocket forwards. The change in a rocket's velocity $\Delta v$ as a function of time $t$ (under a set of simplifying assumptions) is given by the Tsiolkovsky rocket equation:

$$\Delta v(t) = v_e \ln \left(\frac{m_0}{m(t)}\right) - gt$$

where $m_0$ is the initial "wet" mass (fuel + rocket parts + payload), $m(t) = m_0 - \dot{m}t$ is the mass at time $t$, $\dot{m}$ is the fuel burn rate (which we will assume to be constant), $v_e$ is the fuel exhaust velocity, and $g$ is the gravitational acceleration.

You can find the altitude $h$ of the rocket at "burnout" (when all the fuel is used up) by integrating the velocity over time:

$$h = \int_{0}^{T} \Delta v(t) \, dt$$

where $T$ is the total burn time of the rocket, which by conservation of momentum arguments is:

$$T = \frac{m_0 - m_f}{\dot{m}}$$

where $m_f$ is the final "dry mass" of the rocket, once all the fuel is burned.

def delta_v(t):
    """
    This function returns the change in velocity of a rocket based on the Tsiolkovski rocket equation
    
    arguments
    t (time elapsed), float, seconds

    Returns velocity as a float
    """

    if type(t) is not float:
        raise (TypeError(f"time input must be a float"))
    if t<0:
        raise (ValueError(f"Time starts at 0, so must be positive"))

    m=m_o-m_dot*t
    if (m_dot*t)<m_o:
        v=v_e*np.log(m_o/m)-g*t
        assert isinstance (v,float), "velocity is not a float"
        return v
    else:
        return 0
